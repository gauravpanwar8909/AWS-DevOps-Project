# AWS DevOps End-to-End Automation Project

##  Author

Gaurav Panwar

---

##  Project Description

This project demonstrates a real-world DevOps workflow:

* Infrastructure created using Terraform
* Server configured using Shell Script
* Application containerized using Docker
* CI/CD pipeline created using GitHub Actions
* Deployment tested on AWS EC2

---

##  Tools Used

* AWS
* Terraform
* Docker
* Linux
* GitHub Actions

---

##  Goal

To demonstrate complete DevOps automation from infrastructure provisioning to deployment.
